# DevProjects - RSS feed reader website

This is an open source project from [DevProjects](http://www.codementor.io/projects). Feedback and questions are welcome!
Find the project requirements here: [RSS feed reader website](https://www.codementor.io/projects/web/rss-feed-reader-website-atx32j280x)

## Tech/framework used
Built with *php*, *javascript* and *bootstrap*

## Installation
Install php if not installed from [php.net](https://www.php.net/downloads.php).

Install git if not installed from [git-scm.com](https://git-scm.com/downloads).

Run [git clone](https://github.com/salvatorericcardi/feed-reader.git).

Run *php -S localhost:8000*
